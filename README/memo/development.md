# 現在の動き
- oichiを高速化し、欲しいコマ(frame)を得られるようにすることで eichiをたのしくする
- eichiを強化し、欲しいコマからコマ(frame to frame)の動画(movie)を得られるようにすることで、 つぎのoichiがたのしくなる
